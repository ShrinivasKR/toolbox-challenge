Memory!

Objective Of The Game

The object of the game is to collect all EIGHT matching pairs.

Instructions

Users can only flip up to two tiles at the same time. IF both flipped tile do not match, users will be given one second to memorize the image behind the tile and its location. IF users got a matching pair, the paired tiles will stay up until users start the game over. There is no time limits as to clear the objective of the game.
